При використанні INNER JOIN ми отримуємо лише ті рядки, де є збіги в обох таблицях (тобто тільки ті книги, які хоча б раз брали в бібліотеці).
Якщо ж ми змінимо тип з'єднання на LEFT JOIN від таблиці books до borrowed_books, кількість рядків збільшиться (або залишиться такою ж, але не зменшиться). Це відбувається тому, що LEFT JOIN примусово повертає всі рядки з лівої таблиці (books), навіть якщо їх жодного разу не брали (тоді в колонках з borrowed_books буде стояти значення NULL).
